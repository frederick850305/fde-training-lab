# 2-49 页面设计保存区简化

## 目标

优化 `04 页面设计` 底部保存区，不在前端暴露 `projectContext.stepResults.page` 等内部实现信息，并明确该步骤保存的是整体“页面设计”输出成果。

## 本节改动

- `PageDesignView.vue`
  - 底部保存区文案改为面向用户的业务表达：保存页面设计成果。
  - 去掉前端展示的内部字段名。
  - 保存提示改为汇总口径：已保存多少个场景、多少个页面。
  - 页面区域局部保存提示保留在页面区域模块内。
- `App.vue`
  - 页面设计保存后的 summary 改为整体成果口径：包含场景页面清单、导航结构、文件建议和页面区域。

## 验证

- `npm run build` 通过。
