# 2-47 当前页面区域生成与保存按钮

## 目标

明确 `04 页面设计` 中页面区域的操作链路：先在左侧页面清单选中一个页面，再在右侧调用大模型生成该页面的区域，最后由用户点击“保存”写入本地上下文。

## 本节改动

- 将页面区域按钮文案从 `调用大模型生成区域` 改为 `调用大模型生成`。
- 在 `新增区域` 旁边新增 `保存` 按钮。
- 点击 `保存` 会保存当前场景下、左侧选中的当前页面区域设计内容。
- 生成页面区域后不再自动保存，而是提示用户点击 `保存` 写入本地。
- 保存后在页面区域区域内显示：当前页面名称、区域数量和保存时间。

## 验证

- `npm run build` 通过。
