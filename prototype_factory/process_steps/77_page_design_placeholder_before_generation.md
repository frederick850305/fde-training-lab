# 2-77 页面设计生成前占位逻辑

## 本次目标

第 04 步页面设计在点击业务场景卡片“生成”之前，不再自动填充页面清单、当前页面和页面区域。

## 已完成

- `PageDesignView.vue` 移除进入页面设计时自动调用本地规则生成页面设计的 watch。
- 如果没有本地 / 浏览器保存的 `page.md` 数据，页面清单显示“待生成页面清单”占位。
- 如果没有当前页面，右侧显示“待生成当前页面”占位。
- 业务场景卡片“生成”后，才填充页面清单和当前页面基础信息。
- 页面区域不再由业务场景生成时提前填充；即使 LLM 返回页面区域，也会先清空，等待点击“调用大模型生成”。
- 页面区域没有保存数据时显示“待生成页面区域”占位。
- 点击页面区域“调用大模型生成”但还没有当前页面时，给出明确提示。

## 规则

- 已保存的本地 / 浏览器页面设计结果仍会优先恢复。
- 未保存的新场景默认只展示占位，不再使用本地规则提前塞入内容。
