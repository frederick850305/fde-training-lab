# 2-54 页面设计映射去泛化修复

## 目标

修复 `04 页面设计` 中不同业务场景都显示同一个 `VehicleScenarioView.vue` 的问题，并移除页面设计顶部重复的当前场景摘要区域。

## 已完成

- 移除 `04 页面设计` 顶部“当前业务场景页面方案”区域，避免与下方页面清单/当前页面详情重复。
- 修复 `02 场景识别` 页面映射命名逻辑：中文场景标签不再统一退化成 `Scenario`。
- 新增业务词映射：现场调度、任务接收、门岗出入核验、业务状态、异常处理、资源协调、进度跟踪、物料跟踪、预约审批等会生成不同 Vue 文件名。
- `04 页面设计` 新增二次纠偏：如果上一步旧数据或大模型返回 `VehicleScenarioView.vue` 等泛化文件名，会根据当前场景、任务流程、角色和模块重新推导页面文件名。
- `重新生成页面方案` 在 DeepSeek 配置可用时优先调用大模型生成页面设计；不可用或失败时回退本地规则生成。

## 验证

- `npm run build` 通过。
