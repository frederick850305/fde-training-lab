海工制造 APS 原型系统 —— 单文件版（零安装）
============================================

如何运行
--------
直接用浏览器打开本目录下的：

    aps.html

即可使用，无需安装 Node.js，也无需启动任何服务器。
（推荐 Chrome / Edge / Firefox 等现代浏览器；双击文件或拖入浏览器窗口均可）

功能说明
--------
- 包含完整的 WBS、工艺路线、资源、排程、异常等原型页面
- 数据保存在浏览器本地（localStorage），刷新不丢失；清空浏览器缓存即重置
- 纯前端 Mock 原型，不涉及后端服务

常见问题
--------
1. 打开后是空白页？
   - 请确认浏览器为较新版本；部分浏览器对本地 file:// 有严格限制，可改用 360/Chrome/Edge。
2. 想换一台电脑继续看之前的数据？
   - 本地数据随浏览器存储，换设备需重新操作或从源码版导入。
3. 需要修改页面或对接真实接口？
   - 请使用「源码版」压缩包，按其中 README 用 npm 启动。
